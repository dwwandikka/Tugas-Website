PROFIL KELOMPOK CODECRAFTERS

Codecrafters merupakan penggabungan antara 2 kata yakni "Code" dan "crafters". Code yang merujuk pada pemrograman ataupun coding dan crafters yang berarti pembuat atau pengerajin. jadi, Codecrafters memberikan kesan bahwa anggota kelompok ini adalah para ahli dalam bidang kode atau program
